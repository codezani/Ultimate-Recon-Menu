# Ultimate Recon Framework (Windows)

PowerShell-based reconnaissance framework for authorized security testing and bug bounty.

## Features
- Multi-stage recon pipeline
- Resumable per-tool execution
- Detailed per-tool logging
- Proxy support
- DNS resolution with dnsx

## Requirements
- Windows 10/11
- PowerShell 5.1+
- Go (for most tools)
- Python 3.9+ (for some tools)
- Git

## Installation
1. Clone the repository
2. Run the install script: `.\Install-ReconTools.ps1`
3. Run the main script: `.\recon.ps1 example.com`

## Legal
Use only on targets you have explicit written permission to test.
